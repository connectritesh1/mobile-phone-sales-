# Regional Sales Performance Dashboard 📊

A PowerBI dashboard analyzing regional sales data to identify revenue drivers and performance trends. Visualizes key metrics like YoY growth, top product lines, and customer demographics to empower data-driven decision-making and optimize business strategies.

## 🎯 Objective
The primary goal of this project is to provide a comprehensive view of sales performance across different regions. By transforming raw sales data into an interactive dashboard, this project enables stakeholders to quickly identify profitable segments and areas needing improvement.

## 🛠️ Technology Stack
* **Data Visualization & Analysis:** PowerBI
* **Data Processing:** Power Query / DAX

## 📈 Key Metrics Visualized
* **Year-over-Year (YoY) Growth:** Tracking revenue momentum.
* **Top-Performing Product Lines:** Identifying high-margin products.
* **Customer Demographics:** Understanding the target audience.
* **Regional Sales Distribution:** Mapping revenue across territories.
